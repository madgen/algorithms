binpow(first, second // 1)

Performs binary exponentiation on given number. Note: it's a bad implementation. Use built-in Erlang math:pow.

factorial(first)

Returns factorial for given integer

Example

number = 5
fac = Algorithms.Math.factorial number
IO.puts fac
#=> 120

gcd(list1)

Returns GCD of two (or more) integers, based on Euclidean algorithm

Example

numbers = [10, 20]
gcd = Algorithms.Math.gcd numbers
IO.puts gcd  # prints 10

numbers = [10, 20, 25]
gcd = Algorithms.Math.gcd numbers
IO.puts gcd
#=> 5

lcm(list1)

Returns LCM of two (or more) integers, based on GCD

Example

numbers = [4, 6]
result = Algorithms.Math.lcm numbers
IO.puts result
#=> 12
